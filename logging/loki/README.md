### ðŸ”§ Solution: Disable Scalable Mode (Enable Monolithic Mode)

We're running **on-prem with NFS** and want 
a **minimal, single-binary Loki deployment**, 
so **disable** the scalable components entirely.

__Modify the values__ file:

```yaml
loki:
  isDefault: true

  # Disable distributed mode
  commonConfig:
    replication_factor: 1

  storage:
    type: filesystem
    filesystem:
      directory: /var/loki/chunks

  schemaConfig:
    configs:
      - from: 2020-10-15
        store: boltdb-shipper
        object_store: filesystem
        schema: v11
        index:
          prefix: index_
          period: 24h

singleBinary:
  enabled: true

write:
  enabled: false
read:
  enabled: false
backend:
  enabled: false

# PVC (optional if using NFS-backed volume)
persistence:
  enabled: true
  storageClassName: "nfs-client"
  accessModes:
    - ReadWriteOnce
  size: 10Gi
```
